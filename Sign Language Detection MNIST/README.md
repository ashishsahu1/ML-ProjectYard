# Sign Language Predictor using MNIST dataset.

With the help of MNIST Sign Language Image dataset, this script helps to identify sign language alphabets.

We make use of OpenCV, Matplotlib, Pandas, Keras and Numpy libraries of Python to create a model with 92% accuracy. 

Dataset - https://www.kaggle.com/datamunge/sign-language-mnist
